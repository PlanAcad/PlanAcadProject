# PlanAcad

## Instrucciones para Windows

1. Descomprimir **env.zip**
1. En la ventana de comandos ejecutar
    ```
    env\Scripts\activate.bat
    ```
    ```
    cd planacad
    ```
    ```
    python manage.py runserver
    ```
1. La aplicación estará corriendo en **http://127.0.0.1:8000/**
